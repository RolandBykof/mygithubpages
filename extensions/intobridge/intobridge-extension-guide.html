IntoBridge Accessibility Extension

User Guide – Screen Reader Support

Note: This extension is under active development. Bugs are expected. All feedback is very welcome! If you encounter a problem, please report it to digitalisaatioasiantuntija@gmail.com and include a description of what happened.

1. Installation

The extension is loaded as a browser extension in Chrome or Edge using Developer Mode.

Download the extension package (ZIP):
http://rolandbykof.github.io/mygithubpages/extensions/IntoBridge.zip
Extract the ZIP to a folder on your computer before proceeding. The folder should contain manifest.json, content.js, and an icons subfolder.

  1. Download and extract the ZIP file from the link above.
  2. Open your browser and go to the Extensions page: chrome://extensions in Chrome or edge://extensions in Edge.
  3. Enable Developer mode.
  4. Click Load unpacked and select the extracted folder.
  5. The extension now appears in your extensions list. It activates automatically on every game table page.

2. Keyboard Modes

Before using any keyboard commands, make sure your screen reader is in interactive mode (Forms mode or Application mode). Consult your screen reader's documentation for how to activate this mode. Without it, key presses will be intercepted by the screen reader instead of reaching the extension.

The extension has two keyboard modes. Press Z at any time to toggle between them. The screen reader announces the active mode when you switch.

Cards mode (default)

Use this mode to ask information about your hand, the dummy's hand, the current trick, bidding history, and so on.

Keyboard mode

Use this mode when you want to act — bid or play a card. Switch to this mode when it is your turn.

Example: press Z to hear "Keyboard mode." Press Z again to hear "Cards mode."

3. Bidding

Bidding uses two keystrokes: first the level (1–7), then the strain. Switch to Keyboard mode before bidding.

  1. Press a digit 1 through 7. The screen reader confirms, for example: "Level 3?"
  2. Press the strain key. The bid is submitted immediately.

Example: press 3, hear "Level 3?", then press N to bid 3NT.

Strain keys: C = Clubs, D = Diamonds, H = Hearts, S = Spades, N = No Trump.

Special calls require only a single key:

  • P — Pass
  • X — Double, or Redouble if a double is already on the table
  • Escape — Cancel a partially entered bid
If you press the wrong level, press another digit before the strain key to correct it. You have 5 seconds to complete each two-key entry before it times out automatically.

4. Playing Cards

Card play uses two keystrokes: first the suit, then the rank. Switch to Keyboard mode before playing. The extension automatically plays from the correct hand — your own hand or the dummy's if you are declarer and it is dummy's turn.

  1. Press the suit key. The screen reader confirms, for example: "Spade?"
  2. Press the rank key. The card is played immediately.

Example: press S, hear "Spade?", then press A to play the Ace of Spades.

Suit keys: S = Spades, H = Hearts, D = Diamonds, C = Clubs.

Rank keys: A = Ace, K = King, Q = Queen, J = Jack, T = Ten, 9 through 2 = nine through two.

Quick-play with arrow keys

When a trick is already in progress (1–3 cards on the table), the arrow keys play the lowest or highest card of the led suit from the active hand, without needing to specify the suit.

  • Arrow Down — play the lowest card of the led suit
  • Arrow Up — play the highest card of the led suit
Arrow keys are blocked when the table is empty (you would be leading) or when all four cards are already played. If you are void in the led suit, the extension tells you and you must choose a discard manually with the suit and rank keys.

Suit-following enforcement

The extension enforces bridge rules automatically. If you select a suit you are not void in while another suit was led, you receive a warning: "Spade? Warning: must follow Heart." If you confirm the play anyway, it is blocked with the message "Must follow suit. You have Heart."

Claim dialog

When a claim or concede dialog appears, the extension moves focus to it automatically and announces the claim text.

IntoBridge bug — Claim buttons are labelled incorrectly. The Accept and Reject buttons in the Claim dialog have their labels swapped in the IntoBridge interface. To accept a claim, press the button announced as "Reject claim". To reject a claim, press the button announced as "Accept claim". This is a known bug in IntoBridge, not in the extension.

5. Query Commands

Query commands read game information aloud. They are available in two ways:

  • In Cards mode — press the letter key alone, no Alt required.
  • In Keyboard mode — hold Alt and press the letter key.

This means you can always check your hand or the board without switching modes.

Example: in Cards mode, press A to hear "3 Spades: A K Q". In Keyboard mode, press Alt+A for the same result.

6. Full Key Reference

Below are the base keys for all commands. Note that for Query Commands, you use the key alone while in Cards mode, but you must hold Alt along with the key when in Keyboard mode.

Query Commands

  • Z: Toggle between Cards mode and Keyboard mode
  • M: Reset extension memory and re-read game state from screen
  • G (or O): Read my entire hand
  • A: My Spades
  • S: My Hearts
  • D: My Diamonds
  • F: My Clubs
  • T (or L): Read dummy's entire hand
  • Q: Dummy's Spades
  • W: Dummy's Hearts
  • E: Dummy's Diamonds
  • R: Dummy's Clubs
  • P: Current trick — cards on the table in chronological order
  • B: Bidding history
  • X: My direction, board number, vulnerability, and contract
  • V: Vulnerability for the current board
  • C: Trick count — We / They
  • N: Player names at the table

Bidding Commands (Active in Keyboard mode during bidding phase)

  • 1–7 then strain key: Bid level then strain. Strain keys are C (Clubs), D (Diamonds), H (Hearts), S (Spades), and N (No Trump)
  • P: Pass
  • X: Double, or Redouble if applicable
  • Escape: Cancel a partially entered bid

Playing Commands (Active in Keyboard mode during play phase)

  • Suit key then rank key: Play a specific card. Suit keys are S, H, D, C. Rank keys are A, K, Q, J, T, 9, 8, 7, 6, 5, 4, 3, 2
  • Arrow Down: Play lowest card of the led suit (only works when a trick is in progress)
  • Arrow Up: Play highest card of the led suit (only works when a trick is in progress)
  • Escape: Cancel a partially entered card play

7. Getting Started with Robots

If this is your first time using the extension, the easiest way to try it is to play a practice game against three robots provided by IntoBridge. No other players are needed.

  1. Go to https://play.intobridge.com/login and sign in to your IntoBridge account. If you do not have an account yet, register for free on the same page.
  2. On the lobby page, find and activate the button labelled "Just Play — Jump in and play stress-free with three friendly robots". This starts a full bridge game at a private table with three computer-controlled robots.
  3. The extension activates automatically. After a couple of seconds you will hear the board announcement, for example "Board 1. None vulnerable.", confirming the extension is running.
