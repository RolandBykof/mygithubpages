Instructions

This is a fully accessible chess game designed for screen reader users.

  • Moving: Enter your move using algebraic notation (e.g., e4, Nf3, O-O) and press Enter.
  • Promotion: To promote a pawn, add the piece's letter after an equals sign (e.g., a8=Q for Queen, h1=N for Knight).
  • Hotkeys:
    • Alt + R: Repeat the last move.
    • Alt + P: Read the positions of all pieces.
  • Inspection: Use the Read Board button to hear the current situation.
  • Move List: Show Moves opens the game history.
