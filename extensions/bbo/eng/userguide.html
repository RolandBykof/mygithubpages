BBO Accessibility Extension - User Manual

Welcome to the user manual for the Bridge Base Online (BBO) Accessibility Extension. This tool is designed to make the BBO interface accessible for screen reader users by automatically announcing game events and providing keyboard shortcuts to read the table state and play cards.

Download and Installation

You can download the extension package from the following link:

http://rolandbykof.github.io/mygithubpages/extensions/bbo.zim

How It Works

The extension runs quietly in the background while you are on the BBO website. It uses standard ARIA live regions to communicate directly with your screen reader. You do not need to hunt for elements on the screen; the extension interprets the visual game board and translates it into speech.

Automatic Announcements

The following events are announced automatically without any keyboard input required:

  • New Boards: Announces the board number and vulnerability when a new game starts.
  • The Auction: Announces every bid made by any player in real-time.
  • Card Play: Announces every card played to the table.
  • Trick Winners: Announces the winner of the current trick once four cards have been played.
  • Board Results: Announces the final contract result and trick count when the board ends.
  • Dialogs and Claims: Automatically reads tournament pop-ups, robotic announcements, and opponent claims, immediately moving your screen reader focus to the dialog buttons (e.g., Accept/Reject).

Keyboard Shortcuts

Use the following keyboard shortcuts to query the game state at any time. All main commands use the Alt modifier key.

General Game Information

  • Alt + B: Read all current bids in the auction.
  • Alt + P: Read the cards currently on the table (the current trick).
  • Alt + C: Read the current trick count (tricks won by us vs. them).
  • Alt + V: Read the vulnerability of the current board.
  • Alt + X: Read a summary containing the Board number, Vulnerability, and current Contract.
  • Alt + H: Read the help menu (speaks the list of available shortcuts).

Playing Mode (Player & Dummy)

These shortcuts are active when you are actively playing a hand.

Reading Your Own Hand

  • Alt + G: Read all cards in your hand.
  • Alt + A: Read your Spades.
  • Alt + S: Read your Hearts.
  • Alt + D: Read your Diamonds.
  • Alt + F: Read your Clubs.

Reading The Dummy (When visible)

  • Alt + T: Read all cards in the Dummy's hand.
  • Alt + Q: Read Dummy's Spades.
  • Alt + W: Read Dummy's Hearts.
  • Alt + E: Read Dummy's Diamonds.
  • Alt + R: Read Dummy's Clubs.

Playing Cards Automatically

If a card has already been led to the current trick, you can use these shortcuts to automatically play a valid card from the active hand (yours or the dummy's) in the led suit.

  • Alt + Up Arrow: Play your HIGHEST card in the led suit.
  • Alt + Down Arrow: Play your LOWEST card in the led suit.

Kibitzing Mode (Spectator Mode)

If you are watching a table rather than playing, press F2 to toggle Kibitzing mode ON. This changes the hand-reading shortcuts so you can inspect all four compass directions.

  • F2: Toggle Kibitzing Mode ON/OFF.
Important Note on Kibitzing Mode: Please note that the Kibitzing shortcuts work at all times. Theoretically, you could use them to check the opponents' cards even when they are not visually visible to you. However, doing so is strictly against the spirit of Bridge, and there is absolutely no point in abusing it. Please use this feature fairly and responsibly.

Reading Hands in Kibitzing Mode

  • South: Alt + G (All cards) | Suits: Alt + A (Spades), Alt + S (Hearts), Alt + D (Diamonds), Alt + F (Clubs).
  • North: Alt + T (All cards) | Suits: Alt + Q (Spades), Alt + W (Hearts), Alt + E (Diamonds), Alt + R (Clubs).
  • West: Alt + 5 (All cards) | Suits: Alt + 1 (Spades), Alt + 2 (Hearts), Alt + 3 (Diamonds), Alt + 4 (Clubs).
  • East: Alt + 0 (All cards) | Suits: Alt + 6 (Spades), Alt + 7 (Hearts), Alt + 8 (Diamonds), Alt + 9 (Clubs).
