Thinkific-videoiden latausohje

Toista nämä vaiheet jokaiselle videolle

1

Siirry videosivulle

Klikkaa kurssin sivupalkissa haluamaasi videota. Odota että sivu latautuu kokonaan.

2

Kopioi iframe-osoite leikepöydälle

Avaa DevTools F12 → Console ja aja tämä skripti:

Console — pääsivu 
const iframe = document.querySelector("iframe");
const teksti = iframe ? iframe.src : "Iframe ei löydy";
const ta = document.createElement("textarea");
ta.value = teksti;
document.body.appendChild(ta);
ta.select();
document.execCommand("copy");
document.body.removeChild(ta);
console.log("Kopioitu: " + teksti);

Osoite on nyt leikepöydällä. Avaa uusi välilehti ja liitä osoite osoitepalkkiin Ctrl+V.

3

Hae mp4-osoite iframe-välilehdeltä

Kun iframe-sivu on auki uudessa välilehdessä, käynnistä video ensin. Aja sitten Console-ikkunassa:

Console — iframe-välilehti 
const entries = performance.getEntriesByType("resource");
const mp4Url = entries
  .filter(e => /deliveries/.test(e.name) && /\.(mp4|m3u8)/.test(e.name))
  .map(e => e.name.replace(/\.m3u8.*/, "/file.mp4"))[0]
  || "Ei löydy — onko video käynnistetty?";

const ta = document.createElement("textarea");
ta.value = mp4Url;
document.body.appendChild(ta);
ta.select();
document.execCommand("copy");
document.body.removeChild(ta);
console.log("Kopioitu: " + mp4Url);

Mp4-osoite on nyt leikepöydällä.

Huom: Jos console tulostaa "Ei löydy", video ei ole ehtinyt ladata. Käynnistä video uudelleen, odota muutama sekunti ja aja skripti uudelleen.
4

Lataa video

Liitä osoite Ctrl+V joko:

Videolaturi-sovelluksella: liitä osoite URL-kenttään ja paina "Lataa suoraan".

Tai komentorivillä:

Komentorivi 
ffmpeg -i "LIITÄ_MP4_OSOITE_TÄHÄN" -c copy video.mp4
Yhteenveto — toista jokaiselle videolle
