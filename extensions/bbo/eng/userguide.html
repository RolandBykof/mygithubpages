BBO Screen Reader Support - User Manual

Welcome to the user manual for the BBO Screen Reader Support extension. This tool is specifically designed for screen reader users by automatically announcing game events and providing keyboard shortcuts to read the table state.

Work in Progress

Please note that this extension is currently under active development. All feedback, bug reports, and suggestions are highly welcome! You can contact the developer at: digitalisaatioasiantuntija@gmail.com

Game Mode Compatibility

This extension is designed to work at tables within the Casual and Competitive sections.

Please note that the extension does not function within the general "Practice" section games, with the sole exception of Bridge Master, which is supported.

Download and Installation

Since this is an unofficial extension, it must be installed manually using the browser's Developer Mode. Follow these steps:

  1. Download the extension package: http://rolandbykof.github.io/mygithubpages/bbo.zip.
  2. Unzip the bbo.zip file to a permanent folder on your computer.
  3. Open your browser (Chrome or Edge) and go to the extensions page by typing chrome://extensions in the address bar.
  4. In the top right corner, switch the Developer mode toggle to ON.
  5. Click the Load unpacked button that appears.
  6. Select the folder where you unzipped the extension files (the folder containing manifest.json).
  7. The extension "BBO Screen Reader Support" should now appear in your list of active extensions.

Required BBO Settings

For keyboard commands and announcements to work correctly, you must configure your BBO account settings as follows (Account -> Settings):

  • General: "Sound effects" = ON
  • Playing: "Pictures of cards" = OFF
  • Playing: "Confirm bids" = OFF
  • Playing: "Confirm cards" = OFF
  • Playing: "Keyboard entry for bidding / play" = ON

Querying Cards, Bids, and Game Events

Use these Alt shortcuts to check the state of the table at any time:

  • Alt + B: Read all current bids in the auction.
  • Alt + P: Read cards currently on the table (current trick).
  • Alt + C: Read current trick count (Us vs. Them).
  • Alt + V: Read vulnerability of the current board.
  • Alt + X: Read summary (Board #, Vulnerability, and Contract).
  • Alt + H: Read the help menu (list of shortcuts).

Checking Hands

  • Your Hand: Alt + G (All cards) | Suits: Alt + A (Spades), Alt + S (Hearts), Alt + D (Diamonds), Alt + F (Clubs).
  • Dummy's Hand: Alt + T (All cards) | Suits: Alt + Q (Spades), Alt + W (Hearts), Alt + E (Diamonds), Alt + R (Clubs).

Bidding and Playing

Screen Reader Mode and Focus

To use keyboard commands for bidding and playing, your screen reader must be in interactive mode (Forms mode / Focus mode), not browse mode. Additionally, the focus must be on the game area and not, for example, in the chat window.

To find the game area:

  1. Switch your screen reader to interactive mode.
  2. Navigate to the button that says your username.
  3. Press the Tab key once. Your focus is now in the game area, and keyboard commands will function correctly.

Bidding Instructions

Type the level (number) first, then the strain (suit):

  • Numbers 1-7 for the level.
  • S (Spades), H (Hearts), D (Diamonds), C (Clubs), N (No Trump).
  • P (Pass), D (Double), R (Redouble).

Playing Card Instructions

Type the suit first, then the rank:

  • S, H, D, or C for the suit.
  • The rank: A, K, Q, J, T (for 10), or numbers 9-2.
  • Example: Typing SA plays the Ace of Spades.

Quick Selection Shortcuts

  • Alt + Up Arrow: Select the highest card in the led suit from the active hand.
  • Alt + Down Arrow: Select the lowest card in the led suit from the active hand.

Automatic Announcements

The following are announced automatically without any keyboard input:

  • New Boards: Board number and vulnerability.
  • The Auction: All bids in real-time.
  • Card Play: Every card played to the table.
  • Trick Winners: Announced after four cards are played.
  • Results: Final contract and trick count.
  • Dialogs: Pop-ups, robot announcements, and claims are read immediately.

Kibitzing Mode (Spectator)

Press F2 to toggle Kibitzing mode to inspect all four hands:

  • South: Alt + G (All) | Suits: Alt + A, S, D, F.
  • North: Alt + T (All) | Suits: Alt + Q, W, E, R.
  • West: Alt + 5 (All) | Suits: Alt + 1, 2, 3, 4.
  • East: Alt + 0 (All) | Suits: Alt + 6, 7, 8, 9.

Fair Play Note

Using these shortcuts to check opponent cards during active play is against the spirit of Bridge. Please use this feature fairly and responsibly.
